## How to capture (free, on Windows)
1. Open the report in **Power BI Desktop** and go to a page.
2. Press **`Win + Shift + S`** (Snipping Tool) → drag over the report area → it copies to clipboard.
3. Paste into **Paint** → **Save As PNG** with the filename above.
4. *(Or)* press **`Win + G`** (Xbox Game Bar) or use **ScreenToGif** to record `demo.gif`.

## Tips for clean shots
- Apply the **Indigo Noir** theme first (`design/theme.json`) so shots look polished.
- Capture at a consistent size (full report canvas, ~1280×720).
- Clear any accidental slicer selections before capturing (use your Reset button).
